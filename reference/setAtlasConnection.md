# Set Atlas Connection (Deprecated)

**Deprecated.** Use
[`getAtlasConnection()`](https://ohdsi.github.io/Picard/reference/getAtlasConnection.md)
instead.

## Usage

``` r
setAtlasConnection(useKeyring = FALSE)
```

## Arguments

- useKeyring:

  Logical. Passed to
  [`getAtlasConnection()`](https://ohdsi.github.io/Picard/reference/getAtlasConnection.md).

## Value

A WebApiConnection object.
